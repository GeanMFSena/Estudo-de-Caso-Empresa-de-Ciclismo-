# Estudo-de-Caso-Empresa-de-Ciclismo-
Este é o meu estudo de caso sobre uma empresa fictícia de bicicletas com sede na cidade de Chicago, abrangendo os primeiros semestres de 2019 e 2020. Neste projeto, utilizei os conceitos aprendidos no módulo inicial de Análise de Dados do Google, aplicando as principais etapas do processo analítico: Ask, Prepare, Process, Analyze, Share e Act.
