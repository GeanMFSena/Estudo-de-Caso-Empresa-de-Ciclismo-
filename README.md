# Estudo-de-Caso-Empresa-de-Ciclismo-
Este é o meu estudo de caso sobre uma empresa fictícia de bicicletas com sede na cidade de Chicago, abrangendo os primeiros semestres de 2019 e 2020. Neste projeto, 
utilizei os conceitos aprendidos no módulo inicial de Análise de Dados do Google, aplicando as principais etapas do processo analítico: Ask, Prepare, Process, Analyze,
Share e Act.

Perguntas Orientadoras de Negócios: Três perguntas guiarão o futuro programa de marketing:
* Como os membros anuais e os ciclistas casuais usam as bicicletas Cyclistic de forma diferente? 
* Por que os ciclistas casuais comprariam associações anuais da Cyclistic? 
* Como a Cyclistic pode usar a mídia digital para influenciar os ciclistas casuais a se tornarem membros?

Nesse estudo de caso estou encarregado de responder a primeira pergunta : Como os membros anuais e os ciclistas casuais usam as bicicletas Cyclistic de forma diferente? 

Para fazer isso precisamos entender o contexto da empresa e os dados que precisamos buscar e analisar e tudo isso voce pode ver no readme "Estudo de Caso Contexto".
